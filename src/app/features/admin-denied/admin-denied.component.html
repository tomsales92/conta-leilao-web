Acesso negado ao Admin

Você não tem permissão para acessar a área de administração.

Para liberar acesso:

  1. Abra o Supabase → seu projeto → Table Editor → tabela profiles.
  2. Encontre a linha do seu usuário (pelo mesmo e-mail que você usa para logar).
  3. Na coluna role, altere de user para admin e salve.
  4. Faça logout no app e entre de novo.
  5. Tente acessar /admin novamente.

Se as colunas plan e role não existirem na tabela, execute o arquivo supabase_migration_plan_role.sql no SQL Editor do Supabase.
